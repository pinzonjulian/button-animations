The Little Animated Button Gallery

By: Julián Pinzón

This small library is meant to enhance the user's experience by adding animations to buttons that indicate a loading process. Click on a button to see the animation run. The close button is an auxiliary item in this example. The event that makes the button go back to normal should come from a server response.
